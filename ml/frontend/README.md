# Olist ML Frontend

React dashboard for visualizing e-commerce performance analytics and machine learning predictions.

## Prerequisites

- Node.js (v18 or later)
- npm or yarn

## Setup

1. **Navigate to the frontend directory:**
   ```bash
   cd ml/frontend
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

## Running the Dashboard

Start the development server:

```bash
npm run dev
```

The application will be available at [http://localhost:5173](http://localhost:5173) (or the port specified in the terminal).

## Features

- **Late Delivery Risk Analysis**: Real-time risk prediction for orders.
- **Customer Segmentation**: Behavioral analysis and segmentation of the customer base.
- **Interactive Visualizations**: Data-driven insights into sales and performance.
