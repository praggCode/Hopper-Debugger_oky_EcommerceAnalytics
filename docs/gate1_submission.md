# Gate 1 Submission

## Sector
E-Commerce / Retail

## Problem Statement
What are the key drivers of sales performance and customer retention across regions in Brazilian e-commerce, and how can the business optimize pricing, delivery efficiency, and customer targeting to maximize revenue?

## Dataset
Brazilian E-Commerce Public Dataset (Olist)  
Source: https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce  

## Dataset Justification
This dataset contains real-world transactional data including orders, customers, payments, products, and reviews.

It meets the project requirements:
- More than 5,000 rows and multiple meaningful columns
- Contains missing values and inconsistencies
- Requires joining multiple tables
- Supports analysis of sales, delivery, and customer behavior
